# Worldview

The lens. Before news, before analysis — *this*.

## Why

The daily Stratagem risks being shaped by whatever framing journalists choose. This folder is the corrective: a curated, stable view of how power, economies, and states actually work — so each day's edition reads news *through* our lens, not the other way around.

## Structure

- **principles/** — how we think about the world (power, economics, geopolitics, psychology)
- **actors/** — the persistent cast (countries, companies, people)
- **frames/** — interpretive tools (48 Laws, mental models, Art of War)
- **influences/** — the inputs that shape the view (books, podcasts, thinkers)
- **sources/** — who to trust and who to discount in daily news
- **pending-updates.md** — where the daily agent proposes changes for weekly review

## How it is used

Each morning, the scheduled agent reads this folder *before* researching news. It then:

1. Filters news through our concerns, not headline categories
2. Uses existing actor profiles instead of re-discovering each story's players
3. Applies the frames (not the media's framing) to explain behavior
4. Proposes updates to this folder via `pending-updates.md` — never edits directly

## Review cadence

- **Daily**: agent reads, does not edit
- **Weekly**: human reviews `pending-updates.md`, merges what survives scrutiny
- **Monthly**: structural review — are any files now wrong? missing?

## Revision discipline

- Keep files short. Dense beats exhaustive.
- State claims, not summaries. "Germany pays for others' wars" beats "Germany has economic exposure."
- When reality contradicts the file, the file changes — not reality.
- Mark judgment calls `[JUDGMENT: ...]` so they can be challenged without re-reading everything.
